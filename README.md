# City Manager App

**City Manager** is a full-stack JavaScript project for managing and searching cities. Demonstrates CRUD operations, dynamic UI updates, and API testing in a modular and testable way.

## Demo

- Add, search, and delete cities dynamically
- Built-in test interface for API endpoints
- Live feedback for user interactions

## Key Highlights

- **CRUD Functionality**: Create, read, and delete cities using a RESTful API  
- **Dynamic UI Updates**: City list refreshes in real-time after operations  
- **Search Functionality**: Filter cities by name and/or country  
- **API Testing Suite**: Validate endpoints directly in the browser, including invalid requests  
- **Error Handling & Feedback**: Clear user messages for success and failure  
- **Modular Code Structure**: Separation of frontend logic, tests, and server API  

## What I Learned

- Structuring a RESTful API with **Deno**  
- Handling client-server communication using the **Fetch API**  
- Building a dynamic, interactive UI with **vanilla JavaScript**  
- Writing in-browser test scripts for API validation  
- Implementing consistent error handling and user feedback  

## Tech Stack

- **JavaScript** (vanilla, ES Modules)  
- **HTML5**  
- **CSS3** (responsive layout and dynamic styling)  
- **Deno** for backend API  
- **Fetch API** for client-server communication  


## How to Run

1. Install Deno 
1. Run the server: deno run --allow-net --allow-read server.js
3. Open index.html in your browser for main app or test.html to run API tests.
